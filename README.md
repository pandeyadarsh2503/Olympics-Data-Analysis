# 🏅 Olympic Data Analysis

A data analysis project that explores historical Olympic data to uncover trends, patterns, and insights about athletes, events, and regions.

---

## 📌 Project Overview

This project analyzes Olympic datasets to answer key questions such as:

- Which countries dominate the Olympics?
- How has athlete participation evolved over time?
- Which sports/events are most popular?
- Regional performance comparisons

The project uses Python for data preprocessing, analysis, and visualization.

---

## 📁 Project Structure
```
.
├── app.py                # Main application / execution script
├── helper.py             # Helper functions for analysis & visualization
├── preprocessor.py       # Data cleaning and preprocessing logic
├── olympics.ipynb        # Jupyter notebook with exploratory analysis
├── athlete_events.csv    # Dataset containing athlete-level data
├── noc_regions.csv       # Dataset mapping NOC codes to regions
├── requirements.txt      # Python dependencies
└── README.md             # Project documentation
```

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/olympic-data-analysis.git
cd olympic-data-analysis
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```
